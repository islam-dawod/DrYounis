# מרפאת ד״ר תחסין יונס — אתר מעוצב מחדש (Static Redesign)

עיצוב מחדש של האתר לפי מסמך המפרט (edits.docx), על בסיס הזהות החדשה מ־logo.pdf,
תוך שמירה מלאה על התוכן העברי הקיים מ־demo.zip.

## הקבצים
- index.html ................. דף הבית (11 מקטעים, מקטע 6.1)
- about.html ................. אודות ד״ר יונס (6.2)
- treatments.html ............ מרכז הטיפולים לפי קטגוריות (6.3)
- treatment-implant.html ..... תבנית עמוד טיפול — השתלת שיניים (6.4)
- before-after.html .......... גלריית לפני/אחרי עם השוואה אינטראקטיבית (6.5)
- instructions.html .......... מרכז הנחיות עם חיפוש וקטגוריות (6.6)
- instruction-implant.html ... תבנית עמוד הנחיה — לאחר השתלה (6.7)
- contact.html ............... צור קשר + טופס תור (6.8)
- styles.css ................. מערכת העיצוב המשותפת (למפתח)

כל עמוד עצמאי (CSS מוטמע + תמונות base64) — אפשר לפתוח בדפדפן בלחיצה כפולה.
קובץ styles.css מצורף בנפרד עבור המפתח שירצה מערכת עיצוב חיצונית משותפת.
